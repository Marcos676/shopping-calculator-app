# Applicacion de descueto en React
Mini proyecto destinado a satisfacer la necesidad de obtener el precio final ,de por ejemplo un producto, con un porcentaje de descuento aplicado. Útil a la hora de hacer las compras en el super.

## Como usarlo?
1. Realizar un clonado `git clone` `https://github.com/Marcos676/porcent-react-app`
2. Abrir editor de código y hacer en la terminal `npm install`
3. Ejecutar el proyecto usando `npm start`
